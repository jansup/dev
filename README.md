# 산업재해 통계 검색 프로그램

고용노동부/공공데이터포털(데이터포털) 공식 통계를 실시간(요청 시 최신 OpenAPI 명세 경로 재확인)으로 조회하는 웹 프로그램입니다.

## 지원 검색

- 지역별: 고용노동부 산업재해현황 (`15002274`)
- 성별: 한국산업안전보건공단 성별 재해자수 (`15064479`)
- 연령별: 한국산업안전보건공단 연령별 재해자수 (`15064486`)
- 업종별: 한국산업안전보건공단 산업중분류별 발생형태별 재해자수 (`15064493`)

## 주요 기능

- API 명세(`infuser.odcloud.kr/oas/docs`)에서 최신 연도 경로 자동 선택
- 업종(대업종/중업종/규모), 지역, 성별, 연령 조건 검색
- 입력 자동완성(필터 옵션 자동 제시)
- 결과 상위 10개 차트 시각화
- 성별/연령/지역 연도별 추이 조회
- 결과 CSV 다운로드
- 조회 시각/원본 데이터 링크 표시

## 실행 방법

1. 의존성 설치

```bash
npm install
```

2. 공공데이터포털 API 키 설정

PowerShell:

```powershell
$env:DATA_GO_KR_API_KEY="발급받은_서비스키"
```

3. 실행

```bash
npm start
```

4. 브라우저 접속

- `http://localhost:3000`

## API 요약

- `GET /api/sources`: 소스 목록 및 최신 명세 정보
- `GET /api/stats?source=...`: 필터 적용 통계 조회
- `GET /api/trend?source=...&category=...`: 연도별 추이 조회(업종 제외)

## 참고

- API 키는 공공데이터포털에서 해당 데이터 OpenAPI 활용신청 후 발급받아야 합니다.
- 현재 최신 통계 경로는 실행 시점 기준으로 자동 탐색됩니다.
